## Bundle Discount Philosophy

### Standard Discount Tiers:
- **15% Discount:** Light incentive, maintains premium perception
- **20% Discount:** Strong value proposition, encourages purchase
- **25% Discount:** Maximum perceived value, best for complete series
- **30%+ Discount:** Deep discount, use sparingly for promotions

### Psychological Pricing:
- End prices in **.99** or **.95** for perceived value
- Avoid **.00** endings (feels less discounted)
- **$39.99** feels significantly less than **$40.00**

**Target:** Committed readers, gift buyers, collectors

**Discount:** 22% - Strong enough to incentivize, not so deep it devalues individual books

### Tier 2: Core Trilogy Bundle
**Target:** Readers wanting the foundational message without the reference book
**Positioning:** "The Essential Rediscover Collection"

**Target:** Serious students, those wanting comprehensive content
**Positioning:** "The Complete Study Collection"

### Tier 4: Starter & Advanced Bundles
**Target:** Price-sensitive customers, specific interest areas
**Positioning:** "Perfect Starting Point" / "Advanced Exploration"
**Discount:** 17-21% - Entry-level pricing, encourages future purchases

## Implementation Considerations

### 1. **Platform Options**

**Amazon KDP:**
- Amazon doesn't offer native bundle functionality
- Must create separate "bundle" listings manually
- Consider creating a "boxed set" product page
- Use Amazon's "Frequently Bought Together" feature

**Gumroad (Recommended for Bundles):**
- Native bundle support
- Easy to create and manage
- Better for digital bundles
- Can offer instant delivery
- Better analytics

**Your Website:**
- Create bundle product pages
- Link to individual Amazon listings
- Offer Gumroad bundles for instant delivery
- Use discount codes if selling directly

### 2. **Bundle Product Creation**

**For Amazon:**
- Create a new product listing titled: "Rediscover Series: Complete 5-Book Bundle"
- Include all 5 book covers in product images
- Description should highlight bundle value
- Set price to bundle price
- Note: This requires manual fulfillment or print-on-demand bundle

**For Gumroad:**
- Create bundle product
- Add all 5 eBooks to bundle
- Set bundle price
- Enable instant delivery
- Add upsell options

### 3. **Marketing Messaging**

**Complete Series Bundle:**
- "Save $17.96 when you buy the complete series"

- "Perfect gift for truth-seekers"

**Core Trilogy:**
- "The essential Rediscover collection"
- "Start with the foundation - 3 core books"
- "Save $8.98 on the trilogy"

**Starter Bundle:**
- "Perfect introduction to the Rediscover Series"

## Bundle Pricing Psychology

### Price Anchoring
- Always show individual prices alongside bundle price
- Display savings prominently: "Save $17.96!"
- Use strikethrough on individual total

### Scarcity & Urgency
- "Limited time bundle pricing"
- "Complete your collection today"
- Countdown timers for promotional pricing

### Value Perception
- Emphasize "per book" savings
- Example: "Just $6.99 per book (vs. $8.99 individually)"
- Highlight total page count/value
